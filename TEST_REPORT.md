## Обнаруженные и исправленные проблемы

### Проблема 1: Неправильное чтение конфигурации
- **Описание:** Dynaconf не мог правильно прочитать значение `environment` из `env.yaml`
- **Причина:** Использование `ENVIRONMENT` вместо `environment` и неправильный способ получения данных из вложенных словарей
- **Решение:** 
  - Использование `to_dict()` для получения данных из Dynaconf
  - Преобразование ключей окружения в верхний регистр для соответствия структуре YAML
  - Рекурсивное преобразование словарей в `SimpleNamespace` с верхним регистром ключей

### Проблема 2: Несоответствие регистра в log_config.py
- **Описание:** Использование `ENVIRONMENT` вместо `environment`
- **Решение:** Заменено на `environment` для соответствия структуре конфигурации
